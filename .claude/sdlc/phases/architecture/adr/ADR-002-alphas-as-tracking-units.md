# ADR-002: Альфы Essence как единицы отслеживания

## Контекст

SDLC-процессы нуждаются в наблюдаемых артефактах прогресса.
Документо-ориентированный подход плодит устаревающие файлы.
Нужен инвариант, отделённый от конкретных рабочих продуктов.

## Решение

Единица отслеживания прогресса — альфа из OMG Essence.
Определения живут в `catalogs/alphas.md`.
Продвижение альфы требует evidence-артефакта.
Единственный источник истины о состоянии — агент `sdlc-alpha-tracker`.
Другие агенты обращаются к трекеру, не читают `alphas.md` напрямую.

## Последствия

- Состояния альф проверяются централизованно, без рассинхрона.
- Рабочие продукты меняются; альфы остаются инвариантом.
- Переход без evidence отклоняется аудитором.
- Требуется дисциплина: не обходить трекер.

## Альтернативы

1. Таск-трекер как источник истины — отвергнуто, смешивает Work и другие альфы.
2. Документы-чеклисты — отвергнуто, нет механизма инварианта.
