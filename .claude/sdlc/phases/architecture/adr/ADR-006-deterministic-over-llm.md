# ADR-006: Детерминированное приоритетнее LLM

## Контекст

LLM-вызов дорог, медленен и недетерминирован.
Многие проверки SDLC сводятся к структурному сравнению.
Форматеры и линтеры — стандартный детерминированный слой.

## Решение

Любая задача, решаемая скриптом или хуком, не делегируется LLM.
Форматеры и линтеры обязательны для фазы development.
Hook-диспетчер не знает языков; команды читаются из `plugin-config.md`.
LLM-агенты применяются только там, где семантика критична.

## Последствия

- Hooks быстрые и предсказуемые.
- Стоимость проверок минимальна.
- Новые языки поддерживаются через конфиг, не код плагина.
- LLM-агенты остаются для аудита, трекинга, SME-опроса.

## Альтернативы

1. LLM на всё — отвергнуто, дорого и медленно.
2. Только скрипты — отвергнуто, теряется семантический слой аудита.
